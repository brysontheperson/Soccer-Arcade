# Rocket League Arena

Arcade browser remake of Rocket League-style car soccer with:

- A full 2v2 match setup
- Three active bots
- Player boost and dodge controls
- An `Auto Play` toggle that hands your car to the bot AI
- Kickoff countdowns, goals, overtime, and a restart flow

## Run It

Open [index.html](./index.html) in a browser.

## Controls

- `WASD` or arrow keys: drive and steer
- `Shift`: boost
- `Space`: dodge / flip burst
- `T`: toggle auto play
- `R`: restart the match
